# Startup Feasibility Analyzer

A Streamlit web application that helps users analyze whether a startup or engineering project is profitable.

## Features
- Profit Calculation
- Break-even Analysis
- Net Present Value (NPV)
- Payback Period
- Future Value and Present Value
- Depreciation Analysis
- Interactive Charts

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
